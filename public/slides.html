PROFIL SISTEM

Latar Belakang & Deskripsi Proyek

ARCHI-TECH Global Systems adalah rancangan platform *company profile* rekayasa infrastruktur digital, otomatisasi pipeline enterprise, dan zero-trust cloud framework.

Fase UAS membuktikan transisi cetak biru rancangan UTS ke produk perangkat lunak web jadi yang dideploy secara aman dan stabil.

Platform ini terdiri dari 4 rute fungsional: Home (Landing), About (Tim & Sejarah), Services (Layanan & Penawaran), dan Contact (Hubungi).

Modularitas Tinggi

Memisahkan komponen antarmuka terdistribusi agar mudah dalam perawatan kode.

Protokol Zero Trust

Sistem keamanan data andal dari form kontak yang terenkripsi dan tervalidasi.

INTERAKTIVITAS

Logika State & Interaktivitas (useState)

1. Navbar Toggle (Navbar.js)

Mengatur menu drawer pada versi mobile agar dapat dibuka dan ditutup dengan merubah status state Boolean.

2. Form Kendali (Contact.js)

Mengumpulkan input field secara asinkron lewat state objek, memudahkan penanganan pengiriman data form.

3. Loader & Quote (About.js)

Mengelola status pemuatan data (`loading`) dan penyimpanan objek kutipan (`insight`) untuk integrasi API.

// Toggle hamburger menu
const [menuOpen, setMenuOpen] = useState(false);

// Form input terpadu
const [formData, setFormData] = useState({
  fullName: '', email: '', subject: '', message: ''
});

const handleChange = (e) => {
  const { name, value } = e.target;
  setFormData(prev => ({ ...prev, [name]: value }));
};
DATA DINAMIS

Integrasi API & Siklus Hidup (useEffect)

Memasukkan data dinamis secara real-time dari Public API pihak ketiga menggunakan hook `useEffect` pada halaman **About.js**:

Endpoint Pihak Ketiga

Menarik data quotes acak bertema teknologi secara asinkron dari endpoint `https://dummyjson.com/quotes/random`.

Fallback System

Bila permintaan gagal/terputus, catch block merespons dengan menampilkan cache lokal yang aman tanpa merusak tampilan.

useEffect(() => {
  fetch('https://dummyjson.com/quotes/random')
    .then(res => res.json())
    .then(data => {
      setInsight(data);
      setLoading(false);
    })
    .catch(err => {
      setInsight(fallback);
      setLoading(false);
    });
}, []);
RESPONSIVITAS

Responsivitas Layar & Framework CSS

Struktur layout didesain dengan prinsip cairan (fluid) agar ramah di berbagai viewport perangkat pengguna:

Desktop View (1200px+)

Grid 4 kolom visual penuh, interaksi hover aktif, serta penempatan menu navigasi horizontal sejajar logo.

Tablet View (768px-1024px)

Grid kolaps ke 2 kolom secara dinamis, teks paragraf mengecil proporsional demi efisiensi baca.

Mobile View (320px-480px)

Menu tersembunyi di balik tombol Hamburger, kolom bertumpuk vertikal satu arah (single column).

Integrasi CSS: Menghubungkan kenyamanan kerangka responsif Bootstrap 5 dengan ketegasan desain kustom Vanilla CSS di berkas terisolasi.
