PROFIL PERUSAHAAN

Latar Belakang & Deskripsi Proyek

ARCHI-TECH Global Systems adalah platform profil perusahaan (*company profile*) rekayasa infrastruktur digital, otomatisasi sistem, dan cloud terdistribusi.

Tujuan utama pengerjaan UAS ini adalah merealisasikan blueprint rancangan visual UTS menjadi program web utuh yang berjalan di server produksi (*Live Deployment*).

Core Pillar

Meningkatkan keandalan sistem melalui modularitas arsitektur perangkat lunak yang andal.

Zero Trust

Setiap protokol komunikasi dan akses diatur dengan enkripsi ujung-ke-ujung.

KONSISTENSI UI/UX

Komparasi Visual (Desain UTS vs UAS)

Blueprint UTS (Figma) DESAIN

Sketsa kasar tata letak halaman yang telah disepakati pada saat UTS:

  • Grid 4 Kolom di bagian Featured Services.
  • Hero Section gelap dengan teks kontras tinggi.
  • Navbar statis di bagian atas layar desktop.
  • Desain statis form kontak dan peta.
Hasil Realisasi UAS (Web Live) IMPLEMENTASI

Implementasi kode nyata yang berjalan responsif di perangkat komputer/mobile:

  • Grid responsif Bootstrap 5 + CSS Flexbox.
  • Hero Image dinamis dengan performa optimal di Vercel.
  • Navbar dinamis dengan menu toggle & click outside handler.
  • Form interaktif dengan manajemen state terpusat.
STRUKTUR KODE

Arsitektur React.js & Komponen Modular

Aplikasi web dipecah menjadi komponen-komponen fungsional terpisah untuk efisiensi kolaborasi tim dan reusabilitas elemen:

HashRouter Navigation

Memungkinkan routing client-side yang mulus tanpa muat ulang halaman penuh di hosting statis.

Modular CSS

Setiap halaman memiliki file stylesheet tersendiri agar mencegah tumpang tindih gaya elemen (styling isolation).

src/
├── components/ # Reusable Global Components
│ ├── Navbar.js
│ └── Footer.js
├── pages/ # Main Page Views
│ ├── Home.js, About.js
│ ├── Services.js, Contact.js
│ └── NotFound.js
├── styles/ # Stylesheets per View
│ ├── variables.css # Design Tokens
│ └── base.css, home.css, ...
└── App.js # Central Router
STATE MANAGEMENT

Logika State & Interaksi (useState)

// 1. Navbar Mobile Toggle (Navbar.js)
const [menuOpen, setMenuOpen] = useState(false);

// 2. Contact Form State (Contact.js)
const [formData, setFormData] = useState({
  fullName: '', email: '', subject: '', message: ''
});

const handleChange = (event) => {
  const { name, value } = event.target;
  setFormData((current) => ({
    ...current, [name]: value
  }));
};

Controlled Components

Seluruh kolom input form dikendalikan secara langsung oleh state React. Hal ini memastikan konsistensi data sebelum dilakukan pengiriman/submit.

Navbar Click Outside Detection

Menambahkan interaktivitas premium berupa pendeteksian klik di luar menu hamburger untuk menutup navigasi secara otomatis menggunakan kombinasi useRef dan event listener.

EXTERNAL INTEGRATION

Integrasi API Dinamis (useEffect)

Mengambil data dinamis dari API publik eksternal menggunakan hook efek samping (useEffect) untuk diintegrasikan ke halaman secara asinkron:

Asynchronous Fetching

Memanfaatkan native API browser `fetch` dengan penanganan error (`catch`) dan state transisi loading demi menjaga pengalaman pengguna tetap mulus.

Data Source

Mengambil quote acak dari platform public API eksternal yang relevan dengan sains/teknologi.

useEffect(() => {
  fetch('https://dummyjson.com/quotes/random')
    .then(res => res.json())
    .then(data => {
      setQuote(data);
      setLoading(false);
    })
    .catch(err => {
      console.error(err);
      setLoading(false);
    });
}, []);
LAYOUT SYSTEM

Responsivitas & Framework CSS

Desktop View (1200px+)

Layout kolom penuh, grid kartu berdampingan, navigasi bar horizontal, serta rendering media resolusi tinggi.

Tablet View (768px - 1024px)

Penyusutan kolom grid secara proporsional. Layout bertransisi dari 4 kolom menjadi 2 kolom demi kenyamanan keterbacaan.

Mobile View (320px - 480px)

Layout vertikal penuh tunggal. Menu navigasi atas disembunyikan di bawah hamburger menu interaktif.

Penerapan Framework & Custom CSS

Mengombinasikan sistem grid Bootstrap 5 yang responsif dengan fleksibilitas Vanilla CSS kustom (menggunakan Custom Properties di `variables.css`) untuk menjamin konsistensi visual di seluruh modul visual.
